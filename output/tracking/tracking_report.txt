  THEORY:
  The Lucas-Kanade tracker minimises the SSD within a patch W:
    E(u,v) = Σ_{(x,y)∈W} [ I2(x+u,y+v) - I1(x,y) ]²

  Brightness constancy + Taylor expansion gives the OFCE:
    Ix*u + Iy*v + It = 0   for each pixel k in W

  Stacked for all pixels: A*d = -b
  Least-squares solution: d = -(AᵀA)⁻¹ Aᵀb

  AᵀA = [[ΣIx²,  ΣIxIy],   Aᵀb = [ΣIx*It, ΣIy*It]ᵀ
         [ΣIxIy, ΣIy² ]]

  Bilinear interpolation at (x+dx, y+dy):
    I ≈ (1-dx)(1-dy)*I(x,y)   + dx*(1-dy)*I(x+1,y)
      + (1-dx)*dy*I(x,y+1)    + dx*dy*I(x+1,y+1)
